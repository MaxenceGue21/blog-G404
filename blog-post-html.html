Découverte HTML & CSS

J'ai découvert à quoi ressemble le code des sites internet : le HTML et le CSS ! Ce sont les langages de programmation qui permettent qu'un site s'affiche, comme celui que vous voyez en ce moment.

  • Le code HTML contient des balises qui permettent de faire des blocs de texte, des liens, des images et plein d'autres choses ! Il est utilisé pour faire le contenu, et uniquement le contenu de la page WEB.
  • Le code CSS permet de mettre des polices d'écriture spéciales, des couleurs, des espacements : bref, tout ce qui est design ! On reprend le contenu du HTML et on lui applique du style pour le rendre plus jolie pour nos yeux !

https://flukeout.github.io/

Il s'agit d'un site pour apprendre à utiliser les sélecteurs CSS.

Qu'est ce qu'un selecteur en CSS ?

En HTML, nous ciblons ce que nous mettons grâce à des balises qui portent un nom.

Selectionnez les balises en CSS

Grâce aux noms des balises, nous pouvons, en CSS, cibler ces balises pour leur appliquer des règles de style. Par exemple, en HTML, nous allons dire : « cet élément est un titre de niveau 1 », la balise s'appelle h1.

En CSS, dans le code, nous allons chercher cette balise (h1) et nous lui appliquons simplement des règles de style adaptés pour cette balise en question.
